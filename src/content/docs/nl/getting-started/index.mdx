## Vereisten

- `sky` binary - [installeer](/nl/installation/) als je dat nog niet gedaan hebt
- `claude` CLI - installeer via [claude.com/claude-code](https://claude.com/claude-code) en voltooi OAuth (voer `claude` eenmaal uit in de terminal)
- Een GitHub-repository met een webhook gericht op je draaiende Skylence-instantie

## 1. Start de daemon

De installer draaide `sky setup` al automatisch (tenzij je `--no-setup` / `-NoSetup` gaf), dus `~/.sky/` is geïnitialiseerd.

**Aanbevolen - draai als service** (start bij inloggen, auto-restart):

```sh
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/install-service.sh | sh
```

Windows: gebruik `install-service.ps1`. Details in [Als service draaien](/nl/installation/#als-service-draaien-auto-restart).

**Handmatig - voor snelle lokale tests:**

```sh
sky serve
```

De daemon bindt aan `127.0.0.1:3090`. Om GitHub te laten binnenkomen, stel hem bloot via een tunnel (Cloudflare, ngrok of vergelijkbaar).

## 2. Registreer een GitHub-webhook

Laat `sky` de webhook voor je registreren:

```sh
sky webhook register --repo owner/naam --url https://jouw-tunnel.example.com
```

Dit maakt de webhook aan op GitHub en schrijft het HMAC-secret naar `~/.sky/.env`. Start `sky serve` opnieuw op om het secret te laden.

Of registreer handmatig via je repository → Instellingen → Webhooks → Webhook toevoegen:
- **Payload URL:** `https://jouw-tunnel.example.com/webhooks/github`
- **Content type:** `application/json`
- **Secret:** plak de waarde van `SKY_GITHUB_SECRET` uit `~/.sky/.env`

## 3. Maak een workflow aan

Workflows staan in `.sky/workflows/*.sky` in je repository. Een minimaal voorbeeld:

```
⊕meta⊕
name = "hallo"
trigger.github.events = ["issues.labeled"]
trigger.github.label = "sky"
⊕⊕

§begroet§
model = "sonnet"
§§

∆begroet∆
Zeg hallo. Vermeld het issue-nummer: {{issue.number}}.
∆∆
```

Valideer het:

```sh
sky lint
```

## 4. Activeer of voer handmatig uit

Wanneer GitHub een event verstuurt dat overeenkomt met de trigger van de workflow, pikt `sky serve` dit automatisch op.

Om te testen zonder webhook-event:

```sh
sky run hallo --repo /pad/naar/repo --issue 42
```

Bekijk resultaten:

```sh
sky logs
```
