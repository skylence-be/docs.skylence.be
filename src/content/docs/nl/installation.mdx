Skylence wordt geleverd als één statische binary genaamd `sky`. Hij draait als lokale daemon op je eigen machine - geen account, geen cloud.

## Installeren

Eén commando. Detecteert automatisch je OS en architectuur, verifieert de checksum, installeert naar `/usr/local/bin/sky` en draait `sky setup` automatisch.

```sh
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/install.sh | sh
```

Controleer:

```sh
sky --version
```

Sla de automatische setup over met `--no-setup` als je die later zelf wilt draaien.

### Installer-opties

```sh
# Pin op een specifieke release
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/install.sh | sh -s -- --version v0.1.1

# Installeer naar een eigen map (geen sudo nodig)
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/install.sh | sh -s -- --dir ~/.local/bin
```

De installer ondersteunt macOS en Linux op zowel amd64 als arm64.

### Windows

```powershell
irm https://github.com/skylence-be/skylence/releases/latest/download/install.ps1 | iex
```

Draait `sky setup` automatisch aan het einde. Controleer:

```powershell
sky --version
```

Sla auto-setup over met `-NoSetup`.

#### Installer-opties

```powershell
# Pin op een specifieke release
& ([ScriptBlock]::Create((irm https://github.com/skylence-be/skylence/releases/latest/download/install.ps1))) -Version v0.2.0

# Installeer naar een eigen map
& ([ScriptBlock]::Create((irm https://github.com/skylence-be/skylence/releases/latest/download/install.ps1))) -Dir C:\tools\sky
```

Vereist Windows 11 (PowerShell 5.1+). Installeert naar `%USERPROFILE%\.sky\bin\sky.exe` en voegt dat pad toe aan de user-PATH.

## Zelf-update

Eenmaal geïnstalleerd werkt `sky` zichzelf bij uit hetzelfde release-kanaal:

```sh
sky update --check      # meldt of er een nieuwere release beschikbaar is
sky update              # installeert de laatste release (met bevestiging)
sky update --to v0.2.0  # pin op een specifieke tag (rollback of reproduceerbare install)
sky update -y           # installeer zonder bevestiging (CI, scripts)
```

## Als service draaien (auto-restart)

Om `sky` als achtergrondservice te draaien die bij inloggen start en automatisch herstart, gebruik de service-installer (meegeleverd met elke release). Geen `sudo` of admin nodig - alle drie platformen installeren op user-niveau.

### macOS / Linux

```sh
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/install-service.sh | sh
```

- **Linux**: installeert een `systemd --user` unit op `~/.config/systemd/user/sky.service` en zet linger aan zodat hij uitloggen overleeft.
- **macOS**: schrijft een LaunchAgent naar `~/Library/LaunchAgents/be.skylence.sky.plist` met `KeepAlive` op non-zero exit.

Werkmap is `~/.sky`, zodat `sky` `config.yaml` vanzelf oppikt.

Verwijderen:

```sh
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/install-service.sh | sh -s -- --uninstall
```

### Windows

```powershell
irm https://github.com/skylence-be/skylence/releases/latest/download/install-service.ps1 | iex
```

Registreert een user-level Scheduled Task `SkylenceServer` met een `AtLogOn`-trigger en een herstart-interval van 1 minuut.

Verwijderen:

```powershell
& ([ScriptBlock]::Create((irm https://github.com/skylence-be/skylence/releases/latest/download/install-service.ps1))) -Uninstall
```

### Samenspel met `sky update`

Zodra de service geïnstalleerd is, noteert `sky update` het wrapper-type (systemd unit / launchd label / Scheduled Task) in `~/.sky/service.json` en bounced die automatisch na een succesvolle binary-replace - geen gat waarin de oude versie nog in het geheugen zit. Uitschakelen per keer met `sky update --no-restart-service`.

## Verwijderen

Verwijder de `sky`-binary:

```sh
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/uninstall.sh | sh
```

Je data-map `~/.sky` (configs, workflow runs, SQLite-DB) blijft standaard bewaard. Om die ook te verwijderen:

```sh
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/uninstall.sh | sh -s -- --purge
```

Sla bevestigingen over met `-y`:

```sh
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/uninstall.sh | sh -s -- --purge -y
```

Als je naar een eigen map installeerde, geef `--dir` mee:

```sh
curl -fsSL https://github.com/skylence-be/skylence/releases/latest/download/uninstall.sh | sh -s -- --dir ~/.local/bin
```

## Handmatig installeren

Als je liever geen script in `sh` pipet, download dan het archief voor je platform direct.

Controleer de checksum (aanbevolen):

```sh
curl -L -O https://github.com/skylence-be/skylence/releases/latest/download/checksums.txt
shasum -a 256 -c checksums.txt --ignore-missing
```

De regel voor jouw archief moet eindigen op `OK`. Zo niet: stop en download opnieuw - voer de binary niet uit.
